## 基本信息

Sese 是一个支持 Windows、Linux 和 macOS 的跨平台基础库，<br>
跨平台支持以源码级跨平台的方式提供。<br>
设计初衷是为了在保持一定的性能、不变更当前技术栈的情况下，<br>
提供一个开箱即用的 C++ 开发框架。<br>
支持以静态链接或动态链接的方式部署至解决方案中。

注意：框架不支持交叉编译，且仅支持 AMD64 架构的机器。
